02 — Model Specification
✓ Correct Model
Nested Random Effects with Explicit Nesting
# ModelR — best specification
lmer(Glycogen ~ Treatment +
  (1 | Treatment:Rat) +
  (1 | Treatment:Rat:Liver),
  REML = TRUE, data = Rats)
Treatment is a fixed effect. Rat and Liver are nested via explicit colon notation, essential when Rat IDs are non-unique across treatments.
✗ Wrong Model 1
Treatment Misspecified as Random Effect
# ModelR_wrong — INCORRECT
lmer(Glycogen ~ 1 +
  (1 | Treatment/Rat/Liver),
  REML = TRUE, data = Rats)
# Expands to: (1|Treat) + ...
Treatment becomes a random effect. No treatment comparison possible. REML = 234.7 vs. 219.6 for correct model — worse fit.
⚠ Wrong Model 2
Non-unique Rat IDs Ignoring Treatment Nesting
# ModelR_wrong2 — INCORRECT
lmer(Glycogen ~ Treatment +
  (1 | Rat/Liver),
  REML = TRUE, data = Rats)
# Singular fit warning!
Assumes Rat IDs are globally unique. Liver variance collapses to ~0. Singular fit. Residual variance = 42.48 (correct model: 21.17).

07 — Conclusion
No Significant
Treatment Effect

Treatment does not significantly affect Glycogen concentration (F(2,3) = 2.93, p = 0.197). Observed differences between Treat1, Treat2, and Treat3 are not large enough relative to substantial between-rat variability. However, with power below 0.80, this is an inconclusive result — not definitive evidence of no effect.
